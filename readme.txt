使用方法说明

如果是mac，可以直接运行main；
编译的话因为程序用到了计算库Eigen，首先需要保证电脑装了Eigen
和g++，如果你的电脑平时g++能找到Eigen则直接运行makefile，
否则请将makefile中-I后面的路径替换为你电脑中的Eigen路径
（注：mac下如果按照Eigen官网的cmake方法安装则默认路径即为
makefile中的路径）。
输入文件为scene.txt，格式为：
[类型]  {[类别]}
        [属性1]= {}
        [属性2]= {}
        ......
end
其中类别是对于物品和光线而言为需要的输入，物品有平面、球、贝
塞尔曲面，光线有点光源和面光源。
